# AgentPulse

A native macOS menu bar app for monitoring Codex usage, quota and activity.

## Features

- Real-time Codex Monitoring
- Token Tracking
- Cost Tracking
- Quota Monitoring
- 用量中心
- Native macOS Experience
- Light & Dark Mode

## Install

Download `AgentPulse.dmg` from GitHub Releases.

## Build

```bash
swift build -c release --disable-sandbox
```

## Package

```bash
./script/package-app.sh
```

## Privacy

All data comes from local Codex session files.
AgentPulse does not upload data to third-party servers.
